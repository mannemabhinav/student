# Student Performance Analysis using Pandas

## Objective
The objective of this project is to analyze student performance data stored in a CSV file using Python and Pandas. The project demonstrates basic data analysis techniques and data visualization.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab / Jupyter Notebook

## Dataset
The dataset contains the following columns:
- StudentID
- Name
- Department
- Marks

## Tasks Performed
1. Loaded the CSV file using Pandas.
2. Displayed the first few records using `head()`.
3. Examined dataset information using `info()`.
4. Generated statistical summaries using `describe()`.
5. Checked for missing values.
6. Calculated average marks department-wise using `groupby()`.
7. Identified the highest and lowest scoring students.
8. Visualized department-wise performance using bar charts.
9. Visualized student-wise marks using bar charts.
10. Generated insights from the dataset.

## Results
- The highest scoring student was identified.
- The lowest scoring student was identified.
- Department-wise average marks were calculated.
- Student performance trends were visualized through charts.

## Conclusion
This project successfully demonstrated how Pandas can be used for data analysis and how Matplotlib can be used for data visualization. Meaningful insights were extracted from the student performance dataset using grouping, aggregation, and graphical analysis techniques.

## Outcome
Basic data analysis and visualization skills were applied to derive insights from a CSV dataset using Python.
